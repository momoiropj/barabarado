# BarabaraDo

Break it down. Then Do.

BarabaraDo is an AI-powered ToDo app that helps you turn big, messy ideas into small, clear actions.

This project is an experimental product by Space Factory.
Currently under active MVP development.

---

## Concept

Big goals are hard to start.
BarabaraDo breaks them down into realistic first steps.

Write what you want to do.
Choose how you're stuck.
Get small ToDos you can actually act on.
